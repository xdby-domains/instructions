XD.GD Registration Instructions

Step 1: Create a Github account. If you already have a Github account, then use that. Make sure you're doing this step on the Github Website(https://github.com/).

Step 2: Go to https://pastebin.com/raw/Yavy9q6B, and do the keybinds "Ctrl+A", and "Ctrl+C", and then close the Pastebin tab.

Step 3: Go to https://github.com/xdgd-domains/register/fork/, leave everything as is, then click "Create fork". Then press "Add file", then click "Create new file".

Step 4: Name the file "your-domain-prefix-goes-here.html", then in the empty big box that says to put your code into it, click on it, do the keybind "Ctrl+V", then scroll up. A form should pop up.

Step 5: Fill out the form, and hit "Commit changes", then press "Commit changes" again.

Step 6: On the home page of your repository, click "Contribute". A menu should pop up. Click on "Open pull request".

Step 7: In the description, type in "XD.GD_VERIFICATION_your-github-username-goes-here", then click on "Create pull request".


Once you have followed these steps, your domain name should be verified within 1 to 90 days. Once the domain name is verified, we'll try to notify you through email.

Note: XD.GD Domains are only allowed if they abide by our Terms of Service.

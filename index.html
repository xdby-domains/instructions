XD.BY Registration Instructions

Step 1: Create a Github account. If you already have a Github account, then use that. Make sure you're doing this step on the Github Website(https://github.com/).

Step 2: Go to https://pastebin.com/raw/Yavy9q6B, and do the keybinds "Ctrl+A", and "Ctrl+C", and then close the Pastebin tab.

Step 3: Go to https://github.com/xdby-domains/register/fork/, leave everything as is, then click "Create fork". Then press "Add file", then click "Create new file".

Step 4: Name the file "your-domain-suffix-goes-here.html", then in the empty big box, do the keybind "Ctrl+V". A form should pop up.

Step 5: Fill out the form, and hit "Commit changes", then press "Commit changes" again.
